# BirdLife Tracker 🦅

O **BirdLife Tracker** é uma aplicação web progressiva (PWA) desenvolvida especificamente para fotógrafos e observadores de aves que desejam organizar os seus registos de campo de forma rápida e eficiente.

## Funcionalidades Principais

- **Registo de Avistamentos:** Guarde detalhes da espécie, local de observação e equipamento utilizado.
- **Captura Integrada:** Utilize a câmara do seu telemóvel diretamente na aplicação para tirar fotos no momento do avistamento.
- **Funcionamento Offline:** A aplicação funciona sem necessidade de ligação à internet, guardando todos os dados no seu dispositivo.
- **Interface Otimizada:** Design simples e intuitivo, pensado para uso rápido no campo.

## Como Instalar no seu Telemóvel

Esta aplicação foi desenhada para ser utilizada como uma aplicação nativa:

1. Abra o link do projeto no navegador do seu telemóvel (Chrome ou Safari).
2. Toque nas definições do navegador (três pontinhos ou botão de partilha).
3. Selecione a opção **"Adicionar ao Ecrã Principal"** (ou "Instalar App").
4. Um ícone será criado no seu menu de aplicações para acesso rápido.

## Tecnologias Utilizadas

- **HTML5 e CSS3** para a estrutura e design.
- **JavaScript** para a lógica de armazenamento e PWA.
- **LocalStorage** para manter os seus registos seguros no seu dispositivo.
